## What is Shell Scripting?

Shell Scripting is series of commands that instructs an operating system to perform a particular tasks. These commands are then executed by command line interprter as a single program. Instead of typing multiple 
commands on the terminal one by one, you put them into a file and run accordingly which helps in automating a repetitive tasks, or handle a complex workflow. 

As you learn Shell scripting, its good to know about Shell as well. 

Shell sits in between the user and Kernel and acts as a bridge between both of them. It receives the commands via the terminal, interprets and translate them into an instructions that your operating system can
execute. Without shell the computer will not understand the command that you type on the terminal. Some commons shells are Bourne shell, Korn shell, Bourne-Again shell, and C shell.
